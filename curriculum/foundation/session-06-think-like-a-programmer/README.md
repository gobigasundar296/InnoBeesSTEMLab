# Session 6: Think Like a Programmer

This module teaches students the fundamental concepts of programming and
computational thinking. Students learn algorithms, flowcharts, variables,
conditions, and loops — then apply these concepts to solve coding challenges
and debug programs from previous sessions.

**Audience:** Classes 6-9  
**Duration:** 2 hours (extendable to 4 hours with all activities)  
**Learning cycle:** Explore → Understand → Build → Code → Challenge → Present

## What students learn

1. What an algorithm is and how to write step-by-step instructions.
2. How to represent logic visually using flowcharts.
3. What variables are and how they store changing data.
4. How conditions (`if`, `else if`, `else`) make decisions.
5. How loops (`for`, `while`) repeat actions efficiently.
6. How functions organize code into reusable blocks.
7. How to read and debug existing programs systematically.
8. How to trace code execution step by step.
9. How to apply programming concepts to solve real problems.
10. How to think algorithmically about everyday problems.

## Core Path for a two-hour class

If only two hours are available, teach:

1. Algorithms with real-life examples (making tea, crossing road).
2. Flowchart symbols and drawing a simple decision flow.
3. Variables: declaring, assigning, and changing values.
4. Conditions: `if`/`else` with Arduino sensor examples.
5. Loops: `for` loop for LED sequences.
6. Debugging exercise: find and fix bugs in given code.

The coding challenges, function design, and advanced problems continue
in extended time.
